# PTE Academic 考试题型全面详解

> PTE Academic（培生学术英语考试）是一种全程机考、AI评分的标准化英语能力测试，总时长约 **2小时**，总分范围 **10-90分**。
> 考试分为三大部分，共包含 **22种题型**（口语部分已新增RTS与SGD题型），采用独特的**交叉评分**机制。

---

## 📊 考试结构总览

| 部分 | 时长 | 题型数量 | 主要评估技能 |
|------|------|---------|------------|
| **Part 1: 口语与写作** | 76-84 分钟 | 9种 | Speaking + Writing |
| **Part 2: 阅读** | 30-41 分钟 | 5种 | Reading |
| **Part 3: 听力** | 30-57 分钟 | 8种 | Listening |

### 评分体系说明

PTE 采用三种评分方式：

| 评分方式 | 说明 | 适用题型 |
|---------|------|---------|
| **正确/错误评分** | 答对得1分，答错不得分 | 多选题、填空题等客观题 |
| **部分得分评分** | 按多维度给予部分分数 | 写作、口语等主观题 |
| **特征评分 (Trait Scoring)** | 评估发音、流利度、语法等具体语言特征 | 口语和写作任务 |

> ⚠️ **倒扣分题型**：阅读多选题 (MCM-R)、听力多选题 (MCM-L)、听力找不同 (HIW) 三种题型选错会倒扣分！

---

## 🔑 交叉评分机制 (Cross-Scoring)

PTE最核心的特点是**交叉评分**——一个题型可以同时贡献多个技能的分数。理解这一机制对制定备考策略至关重要。

| 题型 | 贡献口语 | 贡献写作 | 贡献阅读 | 贡献听力 | 重要性 |
|------|---------|---------|---------|---------|-------|
| Read Aloud | ✅ 高 | | ✅ 中 | | ⭐⭐⭐⭐⭐ |
| Repeat Sentence | ✅ 高 | | | ✅ 高 | ⭐⭐⭐⭐⭐ |
| Summarize Written Text | | ✅ 高 | ✅ 中 | | ⭐⭐⭐⭐ |
| Essay | | ✅ 高 | | | ⭐⭐⭐⭐ |
| Write from Dictation | | ✅ 高 | | ✅ 高 | ⭐⭐⭐⭐⭐ |
| Summarize Spoken Text | | ✅ 中 | | ✅ 高 | ⭐⭐⭐⭐ |
| R&W Fill in the Blanks | | | ✅ 高 | | ⭐⭐⭐⭐ |

## Part 1: 口语与写作 (Speaking & Writing)

### 1️⃣ Read Aloud (RA) — 朗读短文

| 属性 | 详情 |
|------|------|
| **题目数量** | 6-7 题 |
| **任务描述** | 朗读屏幕上显示的一段文本（约60词） |
| **准备时间** | 30-40 秒 |
| **作答时间** | 40 秒 |
| **贡献技能** | 口语 + 阅读 |
| **难度** | ⭐⭐⭐ |
| **权重** | 口语部分占比约 40%，极为重要 |

**评分标准（每题最高15分）：**

| 评分维度 | 分值 | 评分要点 |
|---------|------|---------|
| **内容 (Content)** | 0-5 | 按正确顺序朗读所有单词，增删改任何词都会扣分 |
| **发音 (Pronunciation)** | 0-5 | 发音清晰准确，重音、语调自然 |
| **口语流利度 (Oral Fluency)** | 0-5 | 语速自然稳定，无不必要停顿、重复或犹豫 |

**备考要点：**
- 准备时间内快速浏览全文，标记难词
- 不要一个词一个词蹦出来读，也不要把一个完整短语拆开
- 犯错不要回头重读，继续往前走
- 注意连读和爆破，虚词要弱读，注意音调（升调降调平调），按照意群有节奏的读
- 可以从任意单词开始，到任意单词结束，建议至少8个单词，名词结尾

**🏆 RA 黄金流程（考试实战）：**

**准备阶段（40秒）：**
1. 先**快速浏览一遍**全文，了解大意
2. 找**停顿点**（逗号、句号、意群分界处）
3. 找**难词**，确认发音
4. **心里默读一遍**，确保流畅

**朗读阶段：**
- ✅ **稳定语速** — 不快不慢，匀速推进，切忌忽快忽慢
- ✅ **自然停顿** — 在意群/标点处停顿，不要在单词中间断开
- ✅ **读到结束** — 不管有没有读错，一直往前读到文章结尾，绝不回头重读

**❓ 遇到不会读的单词怎么办：**
- **按音节猜着读** — 根据拼写猜个大概发音，说错也比不说强
- **快速带过** — 用正常语速一带而过，不要犹豫停顿，AI对流利度扣分比发音更狠
- **绝对不要跳过** — 跳词扣Content分，读错只扣Pronunciation分，读错比不读好
- **不要重读** — 读错了继续往前，回头重读会严重影响Fluency评分
- 💡 记住优先级：**流利度 > 发音准确 > 内容完整**

### 2️⃣ Repeat Sentence (RS) — 复述句子

| 属性 | 详情 |
|------|------|
| **题目数量** | 10-12 题 |
| **任务描述** | 听一个句子（3-9秒），然后精确复述 |
| **贡献技能** | 口语 + 听力 |
| **难度** | ⭐⭐⭐⭐ |
| **权重** | 对口语和听力均有高贡献，公认最重要题型之一 |

**评分标准（每题最高9分）：**

| 评分维度 | 分值 | 评分要点 |
|---------|------|---------|
| **内容 (Content)** | 0-3 | 3分=全部正确复述；2分=≥50%正确；1分=<50%但有部分正确；0分=几乎无内容 |
| **发音 (Pronunciation)** | 0-3 | 发音元音/辅音清晰，重音位置正确 |
| **口语流利度 (Oral Fluency)** | 0-3 | 流畅自然，节奏感好 |

**备考要点：**
- 练习短期记忆，尝试记住句子的含义而非逐词记忆
- 听到句子后立刻开始复述，不要犹豫
- 即使只记住部分也要说出来，不要放弃

**🏆 RS 做题技巧详解（终极满分策略）：**

**🎯 核心原则：** 听意群，记框架，流利复述。不需要逐字还原，抓住关键词和句子结构即可。

#### 一、听的策略（录音播放时）

1. **听意群而非逐词**：把句子分成 2-3 个意群记忆，例如 "The university library / will be closed / during the summer break"
2. **抓句首和句尾**：短期记忆对开头和结尾印象最深（首因效应+近因效应），中间模糊可以猜
3. **边听边在脑中影子跟读**：录音播放时心里跟着默念，加深记忆
4. **快速记笔记（可选）**：写字速度够快可速记 2-3 个关键词首字母，辅助回忆

#### 二、说的策略（复述时）

1. **录音结束后立刻开口**：不要犹豫超过 1 秒，哪怕先说出记住的部分
2. **模仿原句的语调和节奏**：AI 评分很看重语调模式，模仿升降调拿更高发音和流利度分
3. **记住多少说多少**：说出 50% 以上的词就能拿 2 分 Content
4. **不确定的词用同义词替换**：忘了 "significant" 可以说 "important"，流利度保住
5. **绝不重复或自我纠正**：说错了继续往前，回头重说严重扣 Fluency

#### 三、不同水平的保底策略

| 水平 | 策略 |
|------|------|
| **能记住 80%+** | 尽量还原原句，注意语调模仿 |
| **能记住 50%-80%** | 说出记住的词，中间用 and/the/is 等虚词填充保持流利 |
| **只记住 30% 以下** | 把记住的几个词串成一句话说出来，绝不放弃 |
| **完全没听懂** | 说任何相关英文句子，至少拿发音和流利度分（保底1-2分） |

#### 🚨 RS 常见错误

- 试图在脑中翻译成中文再翻回英文 → 直接用英文记忆
- 纠结某个词的准确性导致长时间停顿 → 流利度比准确性更重要
- 说到一半放弃不说了 → 哪怕只记住3个词也要说完
- 用中文谐音记忆 → 效果差且影响发音评分

> 🧠 **备考建议**：每天练习 20-30 句，重点刷高频机经句。推荐用影子跟读法（Shadowing）训练短期记忆能力。

### 3️⃣ Describe Image (DI) — 描述图像

| 属性 | 详情 |
|------|------|
| **题目数量** | 3-4 题 |
| **任务描述** | 描述屏幕上的图表/图片（柱状图、折线图、饼图、表格、流程图、地图等） |
| **准备时间** | 25 秒 |
| **作答时间** | 40 秒 |
| **贡献技能** | 口语 |
| **难度** | ⭐⭐⭐⭐ |
| **权重** | 口语部分占比约 25分，非常重要 |

**评分标准（每题最高15分）：**

| 评分维度 | 分值 | 评分要点 |
|---------|------|---------|
| **内容 (Content)** | 0-5 | 准确描述关键信息：标题、轴标签、最高/最低值、趋势、对比 |
| **发音 (Pronunciation)** | 0-5 | 发音准确清晰 |
| **口语流利度 (Oral Fluency)** | 0-5 | 匀速流畅，不磕巴 |

**备考要点：**
- 使用固定模板：介绍图表类型 → 描述关键数据 → 总结趋势
- 25秒准备时间抓住3-4个关键点
- 不需要描述所有细节，重点突出主要信息

---

**🏆 PTE DI 四大题型【终极模板】**

以下模板可覆盖 95% 的 DI 题，考试直接套用即可，不需要临场思考。

#### 一、 图表类（柱状 / 折线 / 饼图 / 表格）

**模板：**
The chart illustrates information about **[主题]**. Overall, we can see a clear trend and noticeable differences in the data. According to the graph, the highest figure is found in **[最大值+时间/类别]**, at around **[数字]**. In contrast, the lowest value appears in **[最小值]**, at approximately **[数字]**. Additionally, another noticeable point is **[中间值/对比]**. In conclusion, this image provides a clear overview of **[主题]**.

**使用技巧：**
- 只说 2-3 个数字
- 不解释原因
- 数字说不准就用 around / approximately

#### 二、 地图 / 平面图类（Floor plan / Building layout）

**模板：**
The diagram illustrates the layout of **[地点/建筑]**. Overall, the area is clearly divided into different functional zones. At the top, there is **[区域 1]**. In the middle, we can see **[区域 2]**. At the bottom, **[区域 3]** is located. In conclusion, the diagram shows a clear and organized layout.

**使用技巧：**
- 只按 上/中/下 或 左/右 描述
- 不要逐个念物品

#### 三、 流程 / 生命周期类（Process / Cycle）

**模板：**
The diagram shows the process of **[主题]**. Overall, it consists of several stages. First, **[第一步]** takes place. Next, **[第二步]** occurs. After that, **[第三步]** is shown. Finally, the process ends with **[最后一步]**. In conclusion, the diagram illustrates the complete process clearly.

**使用技巧：**
- 狂用 First / Next / After that / Finally 这类连接词
- 不要加细节解释

#### 四、 图片 / 场景类（People / Objects / Situation）

**模板：**
The picture illustrates a scene related to **[主题]**. Overall, the image shows a clear situation. In the foreground, we can see **[主要人物/物体]**. In the background, **[次要元素]** is visible. In conclusion, this picture presents a clear view of **[主题]**.

**使用技巧：**
- 前景 + 背景即可
- 不要编故事、不加个人情绪

---

#### 💡 结尾万能句（所有题通用）
如果时间快到了或者脑子卡壳，直接甩出这句：
> **In conclusion, this image provides a clear overview of the given information.**

---

#### 🚨 考试必记 3 条铁律

1️⃣ **所有 DI 用同一结构**（不要针对每道题换句型，形成肌肉记忆）
2️⃣ **不卡顿比信息准更重要**（内容只要贴边就行，流利度是拿分核心）
3️⃣ **30-40 秒说完立刻停**（千万不要拖延或者停顿太久）

> 🧠 **备考建议**：拿着这四套模板，每天对着不同的图干练 10-15 张，练到脱口而出。

### 4️⃣ Retell Lecture (RL) — 复述讲座

| 属性 | 详情 |
|------|------|
| **题目数量** | 1-2 题 |
| **任务描述** | 听一段学术讲座（约60-90秒），然后用自己的话复述 |
| **准备时间** | 10 秒 |
| **作答时间** | 40 秒 |
| **贡献技能** | 口语 + 听力 |
| **难度** | ⭐⭐⭐⭐⭐ |
| **权重** | 对口语贡献约 10-12分，也贡献听力分数 |

**评分标准（每题最高15分）：**

| 评分维度 | 分值 | 评分要点 |
|---------|------|---------|
| **内容 (Content)** | 0-5 | 涵盖讲座主要观点和支持细节 |
| **发音 (Pronunciation)** | 0-5 | 发音标准 |
| **口语流利度 (Oral Fluency)** | 0-5 | 语速自然流畅 |

**🏆 RL 做题技巧详解（终极满分策略）：**

**🎯 终极目标：** 把听到的实词无脑加在模板中。
**⚖️ 评分偏重：** 流利度（最重要） > 发音 > 内容

#### 一、 黄金模板（熟练背诵 ‼️）

> **结构要求：** 开头1句 + 内容4句 + 结尾1句。**说够 30-35 秒！**

1. **The lecture is about** `_____[主题词]_____`.  *(开头句)*
2. **The speaker argues the statement with supporting details, including** `_____[实词1、2、3]_____`. *(内容句1)*
3. **According to the lecture, it is known that** `_____[实词4、5、6]_____`. *(内容句2)*
4. **It is also known that** `_____[实词7、8、9]_____`. *(内容句3)*
5. **It is clear that** `_____[实词10、11、12]_____`. *(内容句4)*
6. *备用句：It is also clear that `_____[备用实词]_____`.*
7. **In conclusion, the lecture is about** `_____[主题词或最后听到的词]_____`. *(结尾句)*

#### 二、 什么算“实词”（10-15个，越具体越好）

模板中横线处要填入你在听录音时记下的实词，重点抓以下几类：
1. **反复出现的词**（通常就是主题词）
2. **重要连接词后的词**（比如 `But`, `However`, `Although`, `So`, `Because` 后面的词）
3. **慢读、重读的词**（说话者刻意强调的信息）
4. **数据词**（具体的数字、年份、百分比 `percentage`）
5. **开头和结尾句的词**

#### 三、 实战操作指南（怎么把词套进去）

1. **结合模板填词：**
   - **第一句/最后一句：** 尽量填主题词（通常是开头句和文中反复出现的词）。
   - **中间填空句：** 每句填入 **1-3 个内容词**，把听到的实词直接往上套，中间用 `and` 连接。（例：`...details, including apple and banana.`）
2. **灵活控制时长（最重要）：**
   - 取决于你记了多少内容词。没说完就继续套模板说，**不用非得总结（内容 > 结构）**。
   - 如果词记少了，说完了还没到30秒，就**重头再读一遍第一句**，保证时长和流利度！

#### 🚨 注意事项（必看铁律）

1. **必须用纸笔：** 边听边做笔记，快速选词。记录时 **1-3词换一行**（方便口语分句说）。
2. **盖好笔帽！** 会用到纸笔，记完记得盖笔帽，不然笔会没水（考场血泪教训）。
3. **不考逻辑、不考语法：** 模板说完词不够就再说一遍。和DI一样，时刻记得**流利度是最重要的**！即使句子意思不通顺，只要你读得顺滑自信，分数就不低！

### 5️⃣ Answer Short Questions (ASQ) — 简短回答

| 属性 | 详情 |
|------|------|
| **题目数量** | 5-6 题 |
| **任务描述** | 听一个简短的常识性问题，用1-2个词回答 |
| **作答时间** | 10 秒 |
| **贡献技能** | 口语 + 听力 |
| **难度** | ⭐⭐ |
| **权重** | 分值较低，但简单易得分 |

**评分标准：**
- **正确/错误评分**：答对得1分，答错得0分
- 只需说出正确答案词汇即可，不需要完整句子

**备考要点：**
- 积累常识性知识词汇
- 听清问题，直接给出答案
- 不确定时给出最佳猜测，不倒扣分

**🏆 ASQ 做题技巧详解：**

#### 一、答题原则

1. **只说 1-2 个词**：不需要完整句子，例如问 "What do you call the person who flies an airplane?" 直接说 "Pilot"
2. **不要犹豫**：录音结束后 3 秒内给出答案，超时可能不录音
3. **说最常见的答案**：不要说偏门词汇，AI 匹配的是标准答案
4. **听不懂也要猜**：答错不倒扣，沉默 = 白丢分

#### 二、高频考点分类

| 类别 | 常见问题举例 | 答案 |
|------|------------|------|
| **职业** | Who treats sick animals? | Vet / Veterinarian |
| **学科** | What is the study of stars called? | Astronomy |
| **地理** | What continent is Brazil in? | South America |
| **日常常识** | How many days are in a leap year? | 366 |
| **工具/物品** | What instrument measures temperature? | Thermometer |
| **动植物** | What is a baby dog called? | Puppy |

> 🧠 **备考建议**：背诵 ASQ 高频机经 100-150 题即可覆盖绝大部分考题，这是最容易拿分的口语题型。

### 🎯 改革新增题型 1: Respond to a Situation (RTS) — 情景回答

| 属性 | 详情 |
|------|------|
| **题目数量** | 每场考试约 2-3 题 |
| **任务描述** | 面对真实的生活或职场场景（如他人请求建议、提出诉求或沟通），根据情境作出自然回应 |
| **准备时间** | 10 秒 |
| **作答时间** | 40 秒 |
| **贡献技能** | 口语 |
| **难度** | ⭐⭐⭐⭐ |
| **权重** | 重点测试情境理解力、即兴反应能力和真实场景的流利度 |

**评分标准（重点评估实际沟通能力）：**
- **内容 (Content)**：回答是否贴合情景要求，是否解决了情景中提出的问题。
- **发音 (Pronunciation)**：发音清晰准确，语音语调自然。
- **口语流利度 (Oral Fluency)**：交流自然，没有过度犹豫和卡顿。

**🏆 RTS 做题技巧：**

1. **快速带入角色**：在10秒准备时间里，明确“我是谁”、“我在和谁说话”、“需要解决什么问题”。
2. **结构化回答**：
   - 第一句：确认情况/打招呼（如：Hi, I understand you are asking about...）
   - 第二句：给出你的建议或回应核心问题（如：I suggest we could... / Here is what happened...）
   - 第三句：总结/礼貌收尾（如：Let me know if this works for you. / Hope this helps.）
3. **保持自然流利**：像真实说话一样，不用过度追求华丽词汇，能清晰把意思传达即可。

### 🎯 改革新增题型 2: Summarize Group Discussion (SGD) — 总结小组讨论

| 属性 | 详情 |
|------|------|
| **题目数量** | 每场考试约 2-3 题 |
| **任务描述** | 聆听一段由3至5人参与的讨论录音（最多3分钟），随后用自己的话对讨论内容进行口头总结 |
| **准备时间** | 10 秒 |
| **作答时间** | 2 分钟 (120 秒) |
| **贡献技能** | 口语 + 听力 |
| **难度** | ⭐⭐⭐⭐⭐ |
| **权重** | 重点测试多方听力理解、信息统整归纳、逻辑组织以及条理清晰的表达能力 |

**评分标准：**
- **内容 (Content)**：是否涵盖了讨论的核心观点以及不同参与者的主要论点。
- **发音 (Pronunciation)**：发音标准，自然清晰。
- **口语流利度 (Oral Fluency)**：语速自然，表达流畅且富有逻辑。

**🏆 SGD 做题技巧：**

1. **高效记笔记**：录音较长（可达3分钟）且人数多，笔记需按“说话人”或“不同观点”分块记录。
   - 比如画十字或三列，分别记录参与者 A, B, C 的核心观点。
2. **提炼分歧与共识**：小组讨论的核心往往是大家在争论什么，最后达成什么共识（或者没有达成）。
3. **口头总结模板（建议说 1-1.5 分钟）**：
   - 引入：The group discussion mainly focuses on [主题].
   - 观点归纳：Some speakers believe that [观点1], while others point out that [观点2].
   - 补充细节：Additionally, it is mentioned that [补充观点].
   - 总结：Overall, despite the different opinions, the group seems to agree that [共识/结论] / the discussion highlights the complexity of [主题].
4. **注重逻辑连接**：大量使用 However, In contrast, On the other hand, Similarly 等连接词。

### 6️⃣ Summarize Written Text (SWT) — 总结文章

| 属性 | 详情 |
|------|------|
| **题目数量** | 1-2 题 |
| **任务描述** | 阅读一篇短文（约300词），用**一句话**总结（5-75词） |
| **作答时间** | 10 分钟 |
| **贡献技能** | 写作 + 阅读 |
| **难度** | ⭐⭐⭐⭐ |
| **权重** | 对写作分数贡献较高 |

**评分标准（每题最高7分）：**

| 评分维度 | 分值 | 评分要点 |
|---------|------|---------|
| **内容 (Content)** | 0-2 | 2分=涵盖主题和要点；1分=涵盖主题或要点；0分=未涉及 |
| **形式 (Form)** | 0-1 | 1分=一句话且5-75词；0分=不满足格式要求 |
| **语法 (Grammar)** | 0-2 | 语法结构正确 |
| **词汇 (Vocabulary)** | 0-2 | 用词准确恰当 |

**备考要点：**
- **全篇只能有一个句号，可以用多个逗号**（可以用从句、连词组合），超过一句直接Form项0分
- 控制在50-75词，**75词（超过直接0分）**
- 先找出原文主旨和2-3个关键论点，用一个复合句串联
- 右下角会有实时字数统计（word count）

**🏆 SWT 做题技巧详解（终极满分策略）：**

#### 一、黄金做题流程（10 分钟分配）

| 阶段 | 时间 | 任务 |
|------|------|------|
| **阅读原文** | 3 分钟 | 找主旨句 + 标记 2-3 个关键论点 |
| **写总结** | 5 分钟 | 用模板把关键点串成一句话 |
| **检查修改** | 2 分钟 | 检查语法、拼写、字数（50-75词） |

#### 二、万能模板结构

> **[主旨], which [论点1], [连接词] [论点2], and [论点3].**

**实用连接词/从句工具：**
- `which` / `that` — 定语从句
- `while` / `whereas` — 对比
- `because` / `since` / `as` — 因果
- `although` / `despite the fact that` — 让步
- `in addition to` / `along with` — 并列补充
- `resulting in` / `leading to` — 结果

**模板示例：**
> The article discusses [主题], arguing that [核心观点], which is supported by [论据1], while also noting that [论据2], and concluding that [结论].

#### 三、找主旨的技巧

1. **看首段首句和末段末句**：80% 的文章主旨在这两处
2. **找反复出现的关键词**：高频词往往就是核心主题
3. **注意转折词后的内容**：`However`, `But`, `Nevertheless` 后面通常是作者真正的观点

#### 🚨 SWT 致命错误清单

- ❌ **写了两个句号** → Form 直接 0 分，总分归零
- ❌ **超过 75 词** → Form 直接 0 分
- ❌ **少于 5 词** → Form 直接 0 分
- ❌ **直接复制原文大段内容** → 词汇分低，且可能 Content 扣分
- ❌ **加入个人观点** → 只总结原文，不发表看法

> 💡 **保底策略**：如果实在找不到要点，就把原文第一句话改写一下，用 which/and 接两个细节，控制在 50-60 词，至少拿到 Form + Grammar 分。

### 7️⃣ Essay (WE) — 议论文写作

| 属性 | 详情 |
|------|------|
| **题目数量** | 1 题 |
| **任务描述** | 根据给定话题写一篇 200-300 词的议论文 |
| **作答时间** | 20 分钟 |
| **贡献技能** | 写作 |
| **难度** | ⭐⭐⭐ |
| **权重** | 约26分贡献给写作，权重非常高 |

**评分标准（每题最高15分）：**

| 评分维度 | 分值 | 评分要点 |
|---------|------|---------|
| **内容 (Content)** | 0-3 | 3分=完全切题，论点充分；2分=部分切题；0分=完全跑题 |
| **形式 (Form)** | 0-2 | 2分=200-300词；1分=120-199或301-380词；0分=<120或>380词 |
| **语法 (Grammar)** | 0-2 | 句式多样，语法正确 |
| **一般语言范围** | 0-2 | 表达准确且多样 |
| **词汇范围 (Vocabulary)** | 0-2 | 词汇丰富，搭配恰当 |
| **拼写 (Spelling)** | 0-2 | 拼写正确（选择英式或美式拼写并保持一致） |
| **发展、结构和连贯性** | 0-2 | 逻辑清晰，段落分明，衔接自然 |

**备考要点：**
- 使用经典议论文结构：引言 → 正文段1 → 正文段2 → 结论
- 严格控制200-300词
- 准备高频话题模板，但不要生搬硬套
- 注意首尾呼应，段与段之间使用过渡词

**🏆 Essay 做题技巧详解（终极满分策略）：**

#### 一、时间分配（20 分钟）

| 阶段 | 时间 | 任务 |
|------|------|------|
| **审题构思** | 2 分钟 | 确定立场，列出 2 个论点 + 各1个例子 |
| **写作** | 15 分钟 | 按四段结构写完 |
| **检查** | 3 分钟 | 检查拼写、语法、字数 |

#### 二、四段式万能结构

**第一段 — 引言（2-3句，约40词）：**
> In today's modern world, [话题] has become a topic of considerable debate. While some people argue that [观点A], others believe that [观点B]. In my opinion, I strongly agree/disagree that [你的立场].

**第二段 — 正文段1（3-4句，约80词）：**
> To begin with, [论点1]. This is because [原因]. For example, [举例]. Therefore, [小结].

**第三段 — 正文段2（3-4句，约80词）：**
> Furthermore, [论点2]. [展开论述]. For instance, [举例]. As a result, [小结].

**第四段 — 结论（2-3句，约40词）：**
> In conclusion, [重述立场]. It is evident that [概括理由]. [展望/建议句].

#### 三、高分词汇替换表

| 基础表达 | 高分替换 |
|---------|----------|
| important | crucial / vital / paramount |
| many | numerous / a multitude of |
| think | maintain / contend / assert |
| good | beneficial / advantageous |
| bad | detrimental / adverse |
| more and more | an increasing number of |
| because | owing to / on account of |
| but | nevertheless / on the other hand |

#### 四、高频话题分类（机经覆盖 90%+）

| 类别 | 常见话题 |
|------|----------|
| **教育** | 网课 vs 面授、大学是否必要、课外活动 |
| **科技** | AI影响、社交媒体利弊、远程工作 |
| **环境** | 气候变化、个人 vs 政府责任 |
| **社会** | 全球化、贫富差距、老龄化 |
| **生活** | 城市 vs 农村、旅行的意义 |

#### 🚨 Essay 常见扣分点

- ❌ **字数不达标**：低于 200 词或超过 300 词，Form 直接扣分
- ❌ **拼写不一致**：英式美式混用（colour vs color），选一种保持统一
- ❌ **跑题**：没有回答题目问的具体问题
- ❌ **没有分段**：至少分 3-4 段，不分段 Structure 项 0 分
- ❌ **模板痕迹过重**：AI 能检测高度雷同的模板，要适当变换用词

> 🧠 **备考建议**：把 20-30 篇高频机经话题的论点和例子提前准备好。考前每天限时 20 分钟写一篇，重点练习在时间压力下的输出速度和拼写准确性。

## Part 2: 阅读 (Reading)

### 8️⃣ Multiple Choice, Single Answer (MCS-R) — 单选题

| 属性 | 详情 |
|------|------|
| **题目数量** | 1-2 题 |
| **任务描述** | 阅读文本，选择一个正确答案 |
| **贡献技能** | 阅读 |
| **难度** | ⭐⭐⭐ |
| **权重** | 较低 |

**评分标准：** 正确/错误评分，选对得1分，选错得0分。不倒扣。

**🏆 MCS-R 做题技巧：**

1. **先读题目再读文章**：带着问题去阅读，节省时间
2. **排除法**：逐一排除明显错误的选项，缩小范围
3. **注意同义替换**：正确答案往往是原文的改写，而非原词照搬
4. **不要过度推断**：答案必须有原文支持，不要脑补
5. **不确定就选最保守的**：不倒扣，大胆选一个

---

### 9️⃣ Multiple Choice, Multiple Answers (MCM-R) — 多选题

| 属性 | 详情 |
|------|------|
| **题目数量** | 1-2 题 |
| **任务描述** | 阅读文本，选择所有正确答案 |
| **贡献技能** | 阅读 |
| **难度** | ⭐⭐⭐⭐ |
| **权重** | 较低 |

**评分标准：** 部分得分 + **倒扣分**
- 每选对一个 +1分
- 每选错一个 **-1分**
- 最低0分（不会出现负分）

**备考要点：**
- ⚠️ 不确定的选项最好不选，错选要倒扣！
- 仔细排除明显错误的干扰项

**🏆 MCM-R 做题技巧：**

1. **宁少勿多**：因为倒扣分，不确定的选项坚决不选
2. **只选 100% 确定的**：一般正确答案 2-3 个，保守选 1-2 个也比冒险好
3. **警惕绝对化表述**：含 "always", "never", "all", "none" 的选项通常是错的
4. **回原文定位验证**：每个选项都要在原文找到对应依据才选
5. **保底策略**：如果完全没把握，只选一个最像的，至少不倒扣

### 🔟 Re-order Paragraphs (RO) — 段落重排

| 属性 | 详情 |
|------|------|
| **题目数量** | 2-3 题 |
| **任务描述** | 将打乱的段落拖拽成正确的逻辑顺序 |
| **贡献技能** | 阅读 |
| **难度** | ⭐⭐⭐⭐⭐ |
| **权重** | 中等 |

**评分标准：** 部分得分
- 按"相邻配对"计分，每一对相邻段落顺序正确得1分
- 例如5个段落，最多可得4分（4对相邻关系）

**备考要点：**
- 先找首段（通常包含主题引入、背景介绍，无代词指代）
- 找逻辑连接词：therefore, however, moreover, this, these 等
- 注意代词的指代关系

**🏆 RO 做题技巧详解：**

#### 一、找首段的 4 个特征

- ✅ **介绍背景或定义概念**（通常以人名、地名、时间开头）
- ✅ **没有代词指代**（不会以 This, These, It, He, She 等开头）
- ✅ **没有逻辑连接词**（不会以 However, Therefore, Moreover 开头）
- ✅ **语气独立完整**，不依赖前文

#### 二、找配对的 3 大线索

| 线索类型 | 举例 | 作用 |
|---------|------|------|
| **代词指代** | this theory, these findings, it, they | 前一段必须提到被指代的名词 |
| **逻辑连接词** | However（转折）、Therefore（因果）、Furthermore（递进） | 暗示与前段的逻辑关系 |
| **时间/顺序词** | First, Then, Later, Finally | 暗示先后顺序 |

#### 三、实战做题流程

1. **快速浏览所有段落**，找到首段（无代词、无连接词、有背景介绍）
2. **从首段出发**，找与之逻辑最紧密的下一段
3. **用代词验证**：下一段的代词/指代是否对应上一段的名词
4. **逐步拼接**，确保每对相邻段落逻辑通顺
5. **最后通读检查**，看整体逻辑是否连贯

> 🧠 **备考建议**：RO 是阅读部分最难的题型，建议每天练 3-5 题，培养对英文逻辑链的敏感度。

### 1️⃣1️⃣ Reading: Fill in the Blanks (R-FIB) — 阅读拖拽填空

| 属性 | 详情 |
|------|------|
| **题目数量** | 4-5 题 |
| **任务描述** | 从选项中拖拽正确的单词填入文章空格 |
| **贡献技能** | 阅读 |
| **难度** | ⭐⭐⭐ |
| **权重** | 对阅读贡献20+分，非常重要 |

**评分标准：** 部分得分
- 每个空格填对得1分，填错得0分，不倒扣

**备考要点：**
- 注意上下文语境、搭配关系和语法一致性
- 先填确定的空格，再推断不确定的

**🏆 R-FIB 做题技巧：**

1. **先通读全文**：了解大意后再填空，不要上来就填
2. **三步判断法**：
   - ① **词性判断**：空格需要名词、动词、形容词还是副词？
   - ② **语义匹配**：哪个词的意思符合上下文？
   - ③ **搭配验证**：和前后词的固定搭配是否正确？
3. **先确定后犹豫**：先填 100% 确定的空，缩小剩余选项范围
4. **注意干扰项**：拖拽题的选项比空格多，有 1-2 个是多余的干扰项

### 1️⃣2️⃣ Reading & Writing: Fill in the Blanks (RW-FIB) — 阅读与写作下拉填空

| 属性 | 详情 |
|------|------|
| **题目数量** | 5-6 题 |
| **任务描述** | 从下拉菜单中选择正确的单词填入空格 |
| **贡献技能** | 阅读 + 写作 |
| **难度** | ⭐⭐⭐⭐ |
| **权重** | 对阅读贡献25+分，阅读部分权重最高的题型！ |

**评分标准：** 部分得分
- 每个空格选对得1分，选错得0分，不倒扣

**备考要点：**
- 结合词义、词性、搭配来判断
- 注意同义词辨析
- 此题型是阅读的**得分大户**

**🏆 RW-FIB 做题技巧（阅读最重要题型！）：**

1. **下拉菜单逐个浏览**：先看每个空格的 4 个选项，快速排除明显不对的
2. **重点关注固定搭配**：
   - 动词 + 介词：`depend on`, `result in`, `contribute to`
   - 形容词 + 名词：`significant impact`, `substantial evidence`
   - 名词 + 名词：`a lack of`, `a range of`
3. **看空格前后 3-5 个词**：答案线索通常就在这个范围内
4. **每题别花超过 2 分钟**：卡住就先跳过，回头再研究

> 🧠 **备考建议**：大量积累英文高频搭配（collocation），推荐用 Oxford Collocations Dictionary 或相关 App 每天学 10 组。

## Part 3: 听力 (Listening)

### 1️⃣3️⃣ Summarize Spoken Text (SST) — 总结听力文本

| 属性 | 详情 |
|------|------|
| **题目数量** | 1 题 |
| **任务描述** | 听一段60-90秒讲座，写一篇 50-70 词的总结 |
| **作答时间** | 10 分钟 |
| **贡献技能** | 听力 + 写作 |
| **难度** | ⭐⭐⭐⭐ |
| **权重** | 对听力和写作均有贡献 |

**评分标准（每题最高10分）：**

| 评分维度 | 分值 | 评分要点 |
|---------|------|---------|
| **内容 (Content)** | 0-2 | 涵盖主要观点和支持细节 |
| **形式 (Form)** | 0-2 | **50-70词** |
| **语法 (Grammar)** | 0-2 | 语法正确 |
| **词汇 (Vocabulary)** | 0-2 | 用词恰当 |
| **拼写 (Spelling)** | 0-2 | 拼写正确 |

**备考要点：**
- 听录音时做笔记，记关键词
- 控制在50-70词，超出或不足都会扣Form分
- 用自己的话改写，展示语言能力

**🏆 SST 做题技巧详解：**

#### 一、时间分配（10 分钟）

| 阶段 | 时间 | 任务 |
|------|------|------|
| **听录音记笔记** | 60-90秒 | 记关键词和核心论点 |
| **写总结** | 6 分钟 | 用模板把关键词串成 2-3 句话 |
| **检查** | 2 分钟 | 检查拼写、语法、字数 |

#### 二、万能模板

> The lecture discusses [**主题词**]. The speaker mentions that [**要点1**] and [**要点2**]. Furthermore, [**要点3**] is also discussed. In conclusion, the lecture provides an overview of [**主题词**].

#### 三、笔记技巧

- **只记实词**：名词、动词、形容词，不记 the/a/is 等功能词
- **用缩写**：government → gov，environment → env
- **每行 1-2 个词**，方便写作时快速扫视

#### 🚨 SST 注意事项

- ❗ 字数必须在50-70词，超出或不足 Form 项 0 分
- ❗ 可以写多个句子（和 SWT 不同！SWT 只能一句）
- ❗ 不需要覆盖所有内容，抓住 2-3 个主要观点即可

---

### 1️⃣4️⃣ Multiple Choice, Multiple Answers (MCM-L) — 听力多选题

| 属性 | 详情 |
|------|------|
| **题目数量** | 1-2 题 |
| **任务描述** | 听录音后选择所有正确答案 |
| **贡献技能** | 听力 |
| **难度** | ⭐⭐⭐⭐ |
| **权重** | 较低 |

**评分标准：** 部分得分 + **倒扣分**（与阅读多选题相同）

**🏆 MCM-L 做题技巧：**

1. **先读选项再听录音**：录音播放前快速浏览所有选项，带着问题听
2. **边听边做笔记**：记录关键信息，听完后对照选项
3. **保守策略**：同样倒扣分，不确定的不选，只选录音中明确提到的
4. **一般选 2-3 个**：正确答案通常 2-3 个，不要全选也不要只选 1 个

---

### 1️⃣5️⃣ Fill in the Blanks (L-FIB) — 听力填空

| 属性 | 详情 |
|------|------|
| **题目数量** | 2-3 题 |
| **任务描述** | 边听录音边在文本缺失处填入正确的单词 |
| **贡献技能** | 听力 + 写作 |
| **难度** | ⭐⭐⭐ |
| **权重** | 中等 |

**评分标准：** 部分得分
- 每个空格拼写正确得1分，拼写错误0分

**备考要点：**
- 提前浏览文本，预判空格位置可能的词性和含义
- 注意单词拼写的准确性，包括单复数

**🏆 L-FIB 做题技巧：**

1. **录音播放前**：快速通读文本，预判每个空格的词性（名词/动词/形容词）
2. **录音播放时**：用手指跟着文本走，到空格处集中注意力
3. **先写后改**：听到什么先写下来，拼写不确定的录音结束后再修改
4. **注意细节**：单复数（-s/-es）、时态（-ed/-ing）、大小写都影响得分
5. **每个空通常只填一个词**：不要填短语

> 🧠 **备考建议**：重点练习常见学术词汇的拼写，如 environment, government, significant, approximately 等。

---

### 1️⃣6️⃣ Highlight Correct Summary (HCS) — 选择正确总结

| 属性 | 详情 |
|------|------|
| **题目数量** | 1-2 题 |
| **任务描述** | 听录音后选择与内容最匹配的总结 |
| **贡献技能** | 听力 + 阅读 |
| **难度** | ⭐⭐⭐ |
| **权重** | 较低 |

**评分标准：** 正确/错误评分，选对得1分

**🏆 HCS 做题技巧：**

1. **录音前快速扫读所有选项**：找出每个选项的核心差异点
2. **听录音时记主旨和关键词**：不需要记细节，抓大意
3. **排除法**：先排除明显与录音内容矛盾的选项
4. **注意语气和结论**：正确选项通常与录音的整体论调一致

---

### 1️⃣7️⃣ Multiple Choice, Single Answer (MCS-L) — 听力单选题

| 属性 | 详情 |
|------|------|
| **题目数量** | 1-2 题 |
| **任务描述** | 听录音后选择一个正确答案 |
| **贡献技能** | 听力 |
| **难度** | ⭐⭐⭐ |
| **权重** | 较低 |

**评分标准：** 正确/错误评分，选对得1分

**🏆 MCS-L 做题技巧：**

1. **先读题目和选项**：带着问题去听，省时省力
2. **听重点而非细节**：关注说话者的主要观点和结论
3. **不倒扣，大胆选**：不确定也要选一个，不会扣分
4. **警惕同义替换**：正确选项往往是录音内容的改写

---

### 1️⃣8️⃣ Select Missing Word (SMW) — 选择缺失的词

| 属性 | 详情 |
|------|------|
| **题目数量** | 1-2 题 |
| **任务描述** | 听一段录音（最后一个词/短语被嘟声替代），选择正确的缺失词 |
| **贡献技能** | 听力 |
| **难度** | ⭐⭐⭐ |
| **权重** | 较低 |

**评分标准：** 正确/错误评分，选对得1分

**🏆 SMW 做题技巧：**

1. **认真听完整段录音**：缺失词的线索在整段话的语境中
2. **关注嘴声前 2-3 句**：缺失词通常是前文的逻辑结论或总结
3. **判断语义方向**：根据上下文判断缺失词是正面/负面/中性含义
4. **不倒扣，必须选**：哪怕猜也要选一个

---

### 1️⃣9️⃣ Highlight Incorrect Words (HIW) — 找出错误词汇

| 属性 | 详情 |
|------|------|
| **题目数量** | 2-3 题 |
| **任务描述** | 边听录音边阅读文本，点击文本中与录音不同的词 |
| **贡献技能** | 听力 + 阅读 |
| **难度** | ⭐⭐⭐ |
| **权重** | 中等 |

**评分标准：** 部分得分 + **倒扣分**
- 每点对一个错误词 +1分
- 每点错一个正确词 **-1分**
- 最低0分

**备考要点：**
- ⚠️ 不确定的词千万不要点！
- 边听边跟读，同步对照文本

**🏆 HIW 做题技巧详解：**

1. **录音前快速浏览文本**：熟悉内容，让自己能跟上录音节奏
2. **用手指/光标跟踪**：边听边用手指指着文本对应位置，保持同步
3. **只点 100% 确定的**：倒扣分机制，点错一个就抵消一个正确的
4. **错误词的常见类型**：
   - 同义词替换：文本写 "big"，录音说 "large"
   - 近似词替换：文本写 "accept"，录音说 "except"
   - 单复数/时态变化：文本写 "study"，录音说 "studies"
5. **每篇通常 5-7 个错误词**：稳拿 3-4 个就是好成绩

> 🧠 **备考建议**：练习时用 1.0x 速度听录音对照文本，培养“同步跟踪”的能力。

---

### 2️⃣0️⃣ Write from Dictation (WFD) — 听写句子

| 属性 | 详情 |
|------|------|
| **题目数量** | 3-4 题 |
| **任务描述** | 听一个句子，将其完整打字写出来 |
| **贡献技能** | 写作 + 听力 |
| **难度** | ⭐⭐⭐ |
| **权重** | 对写作贡献约19分，对听力贡献约10分，**全场最高分值题型之一！** |

**评分标准：** 部分得分
- **逐词计分**：每个正确拼写且位置正确的单词得1分
- 拼写错误（含单复数、时态形式错误）= 该词0分
- 多写的词不倒扣，但也不加分

**备考要点：**
- 🏆 **得分性价比最高的题型！一定要重点练习**
- 练习听写能力，注意单词拼写细节
- 先听完整句子再写，不要边听边写导致漏词
- 背诵高频WFD句子（机经）

**🏆 WFD 做题技巧详解（全场最高性价比！）：**

#### 一、做题流程

1. **听第一遍**：集中精力听完整句子，理解大意
2. **快速在纸上记关键词**：特别是实词（名词、动词、形容词）
3. **立刻打字**：先把记住的词全部打出来
4. **补充虚词**：根据语法补上 a/the/is/are/in/on 等
5. **检查拼写**：逐词检查拼写，特别注意单复数和时态

#### 二、拼写核心技巧

- **虚词也算分**：the, a, is, are, in, on 这些简单词每个都算 1 分，别漏掉
- **大小写不影响评分**：但句首字母大写是好习惯
- **拼写错 = 该词 0 分**：不确定的词尽量写上，谁知道会不会蒙对
- **句末不需要句号**：句号不影响评分，但加上也无妨

#### 三、记忆技巧

- **复制粘贴大作文 到 WFD输入框里面**：无顺序、字数限制
- **听意群而非逐词**：把句子分成 2-3 个意群记忆
- **先记实词后补虚词**：实词是骨架，虚词可以根据语法补充
- **背机经 = 直接拿分**：WFD 题库重复率极高，背了就是送分

> 🧠 **备考建议**：每天背 10-15 句高频 WFD 机经，考前把 200-300 句高频句全部背完。这是 PTE 考试中最容易通过背题拿分的题型，性价比极高！

## 💡 总复习策略

1. **优先刷高权重题型**：RA、RS、WFD、RW-FIB 四大金刚必须拿稳
2. **善用交叉评分**：一题多得分的题型投入产出比最高
3. **规避倒扣风险**：多选题和HIW不确定就不选
4. **背机经**：特别是WFD和RS的高频题目
5. **练口语流利度**：PTE的AI评分非常看重流利度和发音
6. **模考计时练习**：适应考试节奏，避免超时

## 📚 PTE 题库与机经（重要！）

### PTE 确实有题库

PTE 与雅思最大的区别之一：PTE 是**全程机考**，题目由系统从一个**有限的题库**中随机抽取。这意味着——**题目会反复出现**。考生对真题的回忆整理就是所谓的 **"机经"**。

这也是很多人选择 PTE 而非雅思的核心原因：**可预测性强，备考更有针对性。**

### 作文题库：约20-30篇高频题

关于机构说 "作文只有20篇里面必中一篇"，**基本属实**：

- 目前流通的高频 Essay 话题大约 **20-30篇**
- 培生会**不定期更新题库**（加入新题、下架旧题），但更新频率不高
- 考试时绝大多数情况下会从这些高频题中出题
- ⚠️ 但**不能100%保证必中**，偶尔会出新题
- 💡 建议：提前把这20-30篇的**思路和论点**准备好，不需要背全文

### 各题型机经价值一览

| 题型 | 题库情况 | 背机经价值 | 说明 |
|------|---------|-----------|------|
| **Write from Dictation** | 🔥 最稳定 | 🏆 极高，必背 | 高频句反复出现，逐词背诵 |
| **Repeat Sentence** | 🔥 大量重复 | 🏆 极高，必背 | 高频句约200-300句 |
| **Read Aloud** | ✅ 有重复 | ⭐ 高 | 提前熟悉难词发音 |
| **Retell Lecture** | ✅ 有重复 | ⭐ 高 | 提前了解讲座主题和框架 |
| **Summarize Written Text** | ✅ 有重复 | ⭐ 高 | 提前准备总结思路 |
| **Essay** | ✅ 约20-30篇 | ⭐ 高 | 准备论点思路，不死背 |
| **Describe Image** | ✅ 有重复 | ⭐ 中等 | 熟悉图表类型和数据 |
| **Answer Short Questions** | ✅ 大量重复 | ⭐ 中等 | 背常识性问答 |

### ⚠️ 使用机经的注意事项

1. **用最新版本的机经**：培生大约每隔几个月更新一批题目，过期机经可能失效
2. **作文不要死记硬背**：AI评分能检测模板痕迹过重的回答，理解思路后用自己的话写
3. **WFD 和 RS 是背机经性价比最高的**：这两个题型要求精确还原，背了就直接拿分
4. **选择靠谱的机经来源**：优先选更新及时、社区活跃的平台
5. **机经只是辅助**：真正的英语能力提升才是根本，不要完全依赖机经
