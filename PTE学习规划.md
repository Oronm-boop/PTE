# PTE 学习规划 — 目标35分（澳洲WHV）

## 📋 个人情况分析

| 项目 | 详情 |
|------|------|
| **目标分数** | 35分（澳洲WHV门槛，≈雅思4.5） |
| **当前水平** | 六级460分 |
| **强项** | 阅读、写作 |
| **弱项** | 听力、口语 |
| **备考周期** | 3个月（2026年3月 → 6月） |
| **每日时长** | 1小时 |
| **总学习时长** | 约90小时 |

### 好消息

> 六级460的底子，目标只有35分，**完全够用**。35分在PTE里属于较低门槛，核心挑战不是英语能力，而是**熟悉机考题型和AI评分规则**。只要策略正确，拿下35分问题不大。

### 核心策略

**不求全面提升，只求稳过35分：**
- 阅读写作是你的优势 → 保住基本分，不需要花太多时间
- 听力口语是短板 → **重点突破高权重题型**，不需要每种都精通
- 背机经 → WFD和RS是最快提分手段

## 📅 三个月学习计划

### 第一阶段：熟悉考试 + 打基础（第1-3周）

> 目标：了解所有题型、熟悉考试流程、开始口语基础训练

| 周次 | 每日安排（1小时） | 具体内容 |
|------|----------------|---------|
| **第1周** | 了解题型 + 模考体验 | ① 精读PTE考试题型文档（30min）② 做一套完整模考，了解自己真实水平（周末抽2小时） |
| **第2周** | 口语入门 + 跟读训练 | ① Read Aloud 跟读练习（30min）② 听力精听练习（30min） |
| **第3周** | 口语 + 听写入门 | ① Repeat Sentence 练习（30min）② Write from Dictation 听写练习（30min） |

**本阶段重点：**
- 🎯 每天**跟读英语材料15分钟**（培养语感和流利度，可用BBC/VOA慢速）
- 🎯 做完第一套模考后，明确各项分数，调整后续重点
- 📌 推荐工具：PTE官方练习平台（免费模考）、B站PTE教学视频

---

### 第二阶段：重点题型突破（第4-8周）

> 目标：主攻高权重题型，背机经，稳定核心得分点

#### 每日1小时分配建议

| 时间段 | 内容 | 时长 |
|-------|------|------|
| **口语训练** | Read Aloud / Repeat Sentence（交替） | 25分钟 |
| **听写训练** | Write from Dictation 背机经 | 20分钟 |
| **阅读/写作** | RW-FIB / Essay（每周各2次） | 15分钟 |

#### 各周侧重

| 周次 | 口语重点 | 听力/写作重点 | 阅读重点 |
|------|---------|-------------|---------|
| **第4周** | RA：每天练3篇跟读 | WFD：开始背高频句（每天5句） | RW-FIB 做题 |
| **第5周** | RS：听+复述练习 | WFD：继续背（累计35句） | R-FIB 做题 |
| **第6周** | DI：练习描述图表模板 | WFD：继续背（累计60句） | 模考阅读部分 |
| **第7周** | RA + RS 混合练 | WFD：复习+新增（累计80句） | Essay 准备3-5篇思路 |
| **第8周** | RS 强化（最重要题型） | WFD：复习巩固 | SWT 练习 |

**本阶段重点：**
- 🏆 **WFD机经必须背到80句以上**（这是最快提分的方式）
- 🏆 **RS高频句反复听练**（从机经里选100句重点练）
- 📌 口语核心：不求完美发音，只求**流利不卡顿**（AI很看重fluency）
- 📌 Essay：准备5篇高频话题的论点框架即可，35分不需要写得多好

---

### 第三阶段：模考冲刺 + 查漏补缺（第9-12周）

> 目标：模考找感觉，稳定在35+分，调整考试状态

| 周次 | 每日安排（1小时） | 重点 |
|------|----------------|------|
| **第9周** | 完整模考1套 + 薄弱项专练 | 分析模考各项得分，找出最弱环节 |
| **第10周** | 针对性补弱 + WFD/RS复习 | 保持机经不生疏 |
| **第11周** | 完整模考1套 + 调整 | 模拟真实考试节奏 |
| **第12周** | 考前冲刺 | 复习WFD/RS机经 + 轻松模考 + 调整心态 |

**本阶段重点：**
- 🎯 至少做 **3-4套完整模考**（推荐PTE官方scored practice test）
- 🎯 模考分数稳定在 **38分以上**再去考（留3分余量）
- 📌 考前一周不要学新内容，专注复习已有机经
- 📌 保证睡眠和状态，机考对注意力要求很高

## 🎯 各题型目标分和策略

> 35分策略：不需要每项都好，保证不拉胯即可

| 技能 | 目标分 | 策略 |
|------|-------|------|
| **口语 Speaking** | 35+ | RA保持流利、RS尽量复述50%以上、DI用模板说满 |
| **写作 Writing** | 40+ | 你的强项！Essay正常写+WFD拿满分 → 轻松超标 |
| **阅读 Reading** | 40+ | 你的强项！RW-FIB和R-FIB正常做 → 轻松超标 |
| **听力 Listening** | 30+ | WFD背机经拿分+RS尽力复述 → 补上短板 |


## ⚠️ 35分考生常见误区

1. **❌ 花大量时间背单词** → 六级底子对35分完全够用，不需要额外背
2. **❌ 每种题型平均用力** → 只需主攻RA、RS、WFD、RW-FIB四个高权重题型
3. **❌ 追求完美发音** → 35分只需流利不卡顿，不需要native发音
4. **❌ 忽略机经** → 机经是35分考生的**最大捷径**，特别是WFD
5. **❌ 不做模考就上场** → 至少做3套模考熟悉节奏，避免考场紧张超时
6. **❌ 口语说不出来就放弃** → 即使只说出一部分也有分，空录音才是0分
